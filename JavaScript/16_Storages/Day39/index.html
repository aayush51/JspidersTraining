    Storage

  • We can only Store Strings in browsers storage

    • Local Storage (5MB)

  • localStorage.setItem('key','value')
  • It stores date in browser it lasts till we manually delete it
  • localStorage.clear()
  • It deletes all the data from local storage
  • localStorage.removeItem('key')
  • It deletes the data from local storage

    • Session Storage (5MB)

  • sessionStorage.setItem('key','value')
  • It stores date in browser it lasts till browser is closed or tab is closed or session is closed
  • sessionStorage.clear()
  • It deletes all the data from session storage
  • sessionStorage.removeItem('key')
  • It deletes the data from session storage
