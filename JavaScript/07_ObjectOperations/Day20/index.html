    This Keyword

      Scenarios of this keyword

    1. In Global scope (outside function) this keyword points towards the window object
    2. In Local scope Inside normal function this keyword points towards the window object
    3. In Local scope Inside arrow function this keyword points towards the window object
    4. In Local scope of Object Inside normal function this keyword points towards the current object
    5. In Local scope of Object Inside arrow function this keyword points towards the window object
    6. object -> normal function -> arrow function : arrow function nested inside normal function points towards current object due to lexical scoping ,
      instead of pointing window object it points towards current object
