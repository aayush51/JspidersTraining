Array

  • Declaring Array

    1. way 1 - Array Literal
    2. way 2 - Using Array class(Constructor) and new keyword

  • No fixed size - dynamic memory allocation
  • Can store any data

  • Functions of array

    • push()
      • takes element as argument, adds element at end(last index) of array
    • pop()
      • removes element from end(last index) of array
    • unshift()
      • takes element as argument, adds element at start(start index) of array
    • shift()
      • removes element at start(start index) of array
    • indexOf()
      • takes element as argument , returns index of element
    • at()
      • takes index as argument , returns element present at given index
    • includes()
      • takes element , checks given element present in array or not returns boolean value

    • slice()
