    • array.sort()
      • sort value by first digit or letter of value
It affects original array
    • array.sort(cbf(a,b)=>{})
      • cbf takes two parameters - current value and next value
It returns a value that is less than, equal to, or greater than 0
If the result is less than 0, the current value comes first
If the result is greater than 0, the next value comes first
If the result is equal to 0, the order of the values remains the same
